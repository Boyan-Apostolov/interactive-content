# Revision 
Before we proceed with this lesson, let us see the programming concepts and techniques from the last one. 

# Nested Conditions
```js
if (condition1) {
    if (condition2)
        // body; 
    else
        // body;
}
```

# Complex Conditions with `&&,` `||`, `!` and `()`
```js
if ((x == left || x == right) && y >= top && y <= bottom) {
    console.log(…);
}
```

# Switch-Case Conditions
```js
switch (selector) {
    case value1:
        construction;
        break;
    case value2:
    case value3:
        construction;
        break;
    …
    default:
        construction;
        break;
}
```
