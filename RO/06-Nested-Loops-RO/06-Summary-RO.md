- Ce sunt **Buclele Imbricate**

- Sintaxa și detaliile în ansamblu 

```java live
int n = 5;
for (int r = 1; r <= n; r++) {
    for (int c = 1; c <= n; c++) {
        System.out.print("*");
    }

    System.out.println();
}
```
