## În această lecție ați învățat:

Buclele for sunt folosite atunci când trebuie să executăm un bloc de cod de mai multe ori.

* Sintaxa buclei for este formată din:

``` js
for (initialization; condition; update) {
  //loop's body;
}
```

* Inițializare

    - **Condiție**

    - **Pas**

    - **Corp**
